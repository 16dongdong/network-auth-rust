# 支持说明

Network Auth Rust 是开源工程案例项目，维护节奏以代码质量、安全性和可复现问题为优先。

## 可以提交 Issue 的内容

- 可复现的运行错误、接口错误或安装问题。
- 文档不清晰、命令失效或配置示例不完整。
- 明确的功能建议，并说明使用场景和预期行为。
- 测试缺口、发布脚本问题或平台兼容性问题。

## 提交前请准备

- 操作系统、Rust 版本、MySQL 或 MariaDB 版本。
- 相关命令、错误输出和最小复现步骤。
- 是否修改过配置、schema、前端资源或部署脚本。

## 不适合公开 Issue 的内容

- 真实数据库密码、云厂商密钥、后台 Token 或完整配置文件。
- 生产域名、服务器 IP、日志、数据库导出或用户数据。
- 未修复漏洞的可利用细节。

安全问题请按 [SECURITY.md](SECURITY.md) 私下报告。
